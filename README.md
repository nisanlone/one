\# 游戏研究 AI 分析辅助工具



🎮 本项目是一个面向游戏研发与 AI 应用方向的辅助工具，最初代码由 AI 生成，我在此基础上完成了整体优化和功能完善，包括系统 Prompt 设计、分析结构化输出、Markdown 报告保存和程序模块化重构，使其更适合作为实习作品展示。项目旨在帮助研发团队快速分析游戏机制、玩法设计或剧情文本，并生成专业、结构化的分析报告。



使用该工具时，用户可以将游戏设计文本输入程序，例如：“玩家在塔防游戏中放置防御塔抵御敌人波次进攻”，程序会调用 Moonshot AI 大模型进行分析。AI 会像专业游戏设计师一样理解核心玩法、玩家行为路径和设计逻辑，并输出包括核心玩法理解、玩家体验分析、留存与商业化潜力、潜在风险点以及可执行改进建议的完整报告。分析结果会在终端打印展示，同时自动生成带有时间戳的 Markdown 文件，如 `2025-12-25\_11-32-00\_game\_analysis.md`，方便团队整理、存档和分享。



这个工具的特点在于将 AI 分析能力与游戏研究方法结合，提供高效、标准化的输出，帮助开发者快速发现问题并指导优化设计。程序在实现上注重安全与易用性：API Key 使用系统环境变量存储，调用 Moonshot AI 服务时通过 HTTP 请求获取分析结果；整个流程包括用户输入、调用 AI 分析、打印结果和保存报告四个模块化步骤，结构清晰，易于扩展，例如可增加批量分析、可视化展示或图形界面操作。



作为开发者，我在此项目中承担了以下工作：优化 AI 系统 Prompt，使输出结构化、专业化；封装调用函数，处理异常和 API 调用安全；实现 Markdown 自动保存功能，保证分析结果可追溯；重构程序入口和交互逻辑，提高易用性。这不仅展示了我对 Python 编程的掌握，也体现了我对游戏研究逻辑和 AI 应用的理解和实践能力。



总之，该工具不仅能够快速生成专业分析报告，还可以作为作品展示我的 AI 应用能力和游戏研究思维，为游戏研发团队提供高效的辅助分析手段，也适合作为实习申请作品或个人项目展示，体现技术能力与游戏设计理解的结合。
